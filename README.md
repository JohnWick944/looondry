## 📱 Overview

**Looondry** is a modern e-commerce application designed for shopping premium streetwear t-shirts and custom coffee mugs. Built with HTML, CSS, and JavaScript, this platform offers a seamless user experience for customers seeking quality apparel and merchandise.

### Key Features

- 🎨 **Curated Streetwear Collection** - Hand-picked t-shirt designs from emerging streetwear brands
- ☕ **Custom Coffee Mugs** - Personalized and branded coffee mugs for everyday use
- 🛒 **User-Friendly Shopping Cart** - Easy-to-use checkout process
- 📱 **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- 🔒 **Secure Payment Integration** - Safe transactions for customers
- 📦 **Quick Shipping** - Fast and reliable delivery options
- ⭐ **Customer Reviews** - Real ratings and feedback from buyers

## 🚀 Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- No installation required - fully web-based

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/JohnWick944/looondry.git
   cd looondry
   ```

2. **Open in Browser**
   ```bash
   # Simply open index.html in your web browser
   open index.html
   # or
   start index.html
   ```

3. **Alternative: Live Server**
   ```bash
   # If you have Python installed
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```
